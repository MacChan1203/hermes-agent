# Hermes Agent（日本語版）

Hermes Agent は、自然文でターミナル操作を扱えるように拡張した、個人開発用の CLI エージェントです。  
このリポジトリは、既存の Hermes をベースにしつつ、**Macちゃんが ChatGPT と一緒に改良した派生版**です。

---

## 概要

この Hermes Agent では、通常の CLI 操作に加えて、以下のような機能を強化しています。

- 自然文からコマンドへの簡易変換
- 作業ディレクトリ（CWD）の保持
- 前回コマンドの記憶と再実行
- 安全モードによる危険操作の抑止
- 読み取り系タスク向けの自律モード
- 結果をまとめる要約表示

たとえば、以下のような自然文を入力できます。

- `今どこ？`
- `親フォルダに戻って`
- `gitの状況を見て`
- `このプロジェクトの状態を見て`
- `最近のコミットを見せて`
- `Python環境を確認して`

---

## この版の特徴

### 1. 自然文コマンド変換

よく使う操作について、自然文からコマンドへ簡易変換できます。

例:

- `今どこ？` → `pwd`
- `一覧見せて` → `ls`
- `差分見せて` → `git diff`
- `履歴見せて` → `git log --oneline -n 5`
- `前回のコマンドもう一回` → 前回コマンドの再実行

---

### 2. CWD の保持

Hermes Agent は、通常のシェルとは別に**内部の作業ディレクトリ状態**を保持します。  
これにより、自然文での連続操作でも文脈を保ったまま動作できます。

---

### 3. 前回コマンドの記憶

直前に実行したコマンドを記録し、確認や再実行ができます。

例:

- `前回のコマンド`
- `前回のコマンドもう一回`

---

### 4. 安全モード

危険性のある操作や、意図しない破壊的操作を避けるため、安全モードを入れています。

現在は以下のような操作をブロックします。

- 削除系操作  
  例: `rm`, `rm -rf`
- Git の書き込み系操作  
  例: `git push`, `git commit`, `git reset --hard`

読み取り系の確認コマンドは許可します。

- `git status`
- `git diff`
- `git log`
- `pwd`
- `ls`

---

## 自律モード

### 自律モード v1
複数の確認コマンドを順番に実行します。

### 自律モード v2
複数ステップ実行に加え、最後に簡単な要約を返します。

### 自律モード v3
現在地が Git リポジトリ外でも、`ls` の結果を見て候補ディレクトリを探し、必要に応じて**自動で移動してから**続きを実行します。

### 要約強化版
Git 系タスクでは以下のような情報を返せます。

- 現在地
- Git リポジトリかどうか
- 変更状態
- 変更ファイル数
- 最新コミット

Python 系タスクでは以下のような情報を返せます。

- 現在地
- Python パス
- Python バージョン

---

### 自律モード v5
依頼文に含まれるキーワードをもとに、候補ディレクトリを選ぶ理由もあわせて表示します。

例:
- `hermesの状態を見て`
- `claudeの状態を見て`

### 自律モード v4
依頼内容に応じて、候補ディレクトリの優先順位を切り替えます。

例:
- `hermesの状態を見て` → `hermes-agent` を優先
- `claudeの状態を見て` → `everything-claude-code` を優先

これにより、ホームディレクトリから実行しても、対象に合ったリポジトリへ自律的に移動して確認できます。


### 自律モード v6
候補ディレクトリを固定順ではなく、スコアリングで比較して選ぶように改善しました。

評価要素:
- 依頼文に含まれるキーワードとの一致
- ディレクトリが実在すること
- Git リポジトリであること

例:
- `claudeの状態を見て` → `everything-claude-code` を高スコアで選択
- `hermesの状態を見て` → `hermes-agent` を高スコアで選択

出力例:
```text
🤖 自律判断: "everything-claude-code" が最有力候補でした（score=14 / 理由: 依頼文に "claude" が含まれる, 既存ディレクトリである, Git リポジトリである）。/Users/hide/everything-claude-code に移動して続行します


### 自律モード v7
候補ディレクトリの比較結果を表示するように改善した。

#### 変更点
- 最有力候補だけでなく、次点候補も表示
- score と理由を併記
- 自律判断の比較過程が分かるようにした

#### 効果
Hermes Agent の判断がより透明になり、比較ベースで対象を選ぶ自律エージェントとしての性質が強まった。


### 自律モード v8
依頼内容を簡単に分類し、目的に応じて観察コマンドを切り替えるように改善しました。

対応例:
- `最近のコミットを見せて`
- `差分を見て`
- `Python環境を確認して`
- `このリポジトリを見て`

モード例:
- 履歴確認モード
  - `git log --oneline -n 5`
- 差分確認モード
  - `git status`
  - `git diff --stat`
- Python確認モード
  - `which python3`
  - `python3 --version`
- 概要確認モード
  - `git status`
  - `git log --oneline -n 3`

これにより、Hermes Agent は対象リポジトリを選ぶだけでなく、依頼の種類に応じて「何を見るべきか」も切り替えられるようになりました。
